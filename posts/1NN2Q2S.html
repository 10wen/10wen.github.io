ES6-11特性

ES6特性

let

  • 不能重复声明变量
  • 块级作用域
  • 不存在变量提升
  • 不影响作用域链

const

  • 常量需要赋初值
  • 常量的值不能修改
  • 块级作用域
  • 数组、对象的值可以修改，因为引用地址不变

变量的解构赋值

模板字符串

对象的简化书写

箭头函数

  • 适合与this无关的
  • this 是静态的，始终指向函数声明时所在的作用域的this，call/apply/bind 都无法改变
  • 箭头函数不能作为构造器实例化对象，即不能使用new操作符
  • 不能使用 argument 对象

函数参数默认设置

rest参数

  • rest 参数须放最后 func(a, b, ...args)

扩展运算符

Symbol

  • 确保对象属性使用唯一标识符，不会发生属性冲突

迭代器

生成器函数

Promise

Promise 封装读取文件

Promise 封装Ajax

Promise.prototype.then 链式调用

  • then() 返回一个 Promise 对象

  • 对象的状态由回调函数执行结果决定

  • 多次Ajax请求

  • 解决回调地狱

Set 集合

Map

  • 键值对，键不限于字符串，可以任意类型

class

Number 的扩展

模块化

  • 防止命名冲突
  • 代码复用
  • 高维护性
